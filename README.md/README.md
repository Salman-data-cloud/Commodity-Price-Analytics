# Bangladesh Commodity Price Intelligence System

Real-time monitoring of 14 critical commodities for food security, inflation control, and import planning.

## Features
- Monthly + Annual price data (1960–2025)
- Categorized: Energy, Food, Industry, Fertilizer
- PostgreSQL database: `final_bd_commodities`
- Ready for Power BI + Prophet forecasting
